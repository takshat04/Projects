EventMatch 🎉

EventMatch is a smart and interactive event discovery platform designed to transform how users explore, plan, and book events. With swipe-based exploration, personalized recommendations, and real-time features, EventMatch offers a fun, social, and seamless event experience.

🚀 Features

- 🔍 **Swipe-Based Discovery** – Intuitive Tinder-style swipe interface for browsing events
- 🧠 **Smart Recommendations** – LightFM-based hybrid model using user-event interactions
- 🎭 **Genre & Organizer Filtering** – Personalize event suggestions by interest or host
- 🧑‍🤝‍🧑 **Group Matching** – Discover and attend events with friends or interest-based groups
- 🗺️ **360° Venue Previews** – Visual venue tours for immersive planning
- 🎞️ **Short Event Reels** – Quick video previews to make decisions easier
- 📅 **Shared Event Plans** – Collaboratively plan outings with others
- 📊 **Real-Time Polling** – Poll friends for time, venue, or event preferences
- 🎟️ **Integrated Ticketing** – In-app booking experience like BookMyShow
- 🕹️ **Gamified Exploration** – Engage users through badges, levels, and discovery rewards

🧠 Tech Stack

- **Recommendation Engine**: LightFM (BPR Loss, hybrid model)
- **Backend**: Python, Pandas, Numpy
- **Frontend**: React.js / Flutter (optional)
- **Database**: SQLite / PostgreSQL
- **Visualization & Filtering**: Custom filters by genre, location, organizer

📊 Evaluation Metrics

- Precision@k, Recall@k
- Hit Rate (HR), NDCG

📈 Impact

EventMatch bridges the gap between users and organizers, making event discovery more engaging, social, and tailored. It combines the excitement of exploration with AI-driven personalization and smooth ticketing in one unified experience.
