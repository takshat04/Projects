📊 Netflix Analytics Dashboard – Tableau Project

This project showcases a visually interactive **Netflix Analytics Dashboard** built using **Tableau**, aimed at exploring global content trends across Netflix’s catalog.

The dashboard is developed using a dataset containing information about over **8,800 Netflix titles** across **86 countries**. It provides data-driven insights into:

- Content distribution by **type** (Movies vs. TV Shows)
- Trends across **genres**, **ratings**, and **release years**
- Top contributing **directors**
- **Country-wise** content availability and trends

The goal is to help understand Netflix’s content strategy, viewer preferences, and global content footprint through rich and dynamic visualizations.
